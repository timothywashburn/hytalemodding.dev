---
title: Template Hytale API
description: Template for documenting Hytale's API.
---

This is a template for creating documentation for Hytale's API.

The directory name should not contain the text "API", and should be named after the API being documented, ex. `block` for the block API.

Use the top-level `index.mdx` file to provide an overview of the API, offering a description, how its architecture is designed, and how to use it. Then, use the provided template files to create documentation for the API:

A sample structure for the `block` API is provided below:

```
hytale-api/
  some-api-name/           # Name of the API
    index.mdx              # Overview of the API
    class1.mdx             # Documentation for class1
    interface1.mdx         # Documentation for interface1
    event1.mdx             # Documentation for event1
    enum1.mdx              # Documentation for enum1
    example1.mdx           # Usage example1
```
